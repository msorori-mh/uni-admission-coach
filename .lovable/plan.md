
## خطة: تحسينات ذكية لتحويل المنصة إلى أداة ضرورية

### تحليل الوضع الحالي

| الميزة المطلوبة | الحالة |
|---|---|
| محاكاة اختبارات مع عداد زمني | موجود بالفعل (90 دقيقة + 2 دقيقة/سؤال) |
| لوحة المتصدرين "أين ترتيبك؟" | موجودة بالفعل مع فلتر التخصص |
| تحليل الأداء | موجود جزئياً (يحتاج تحسين) |
| بنك أسئلة ذكي بتصنيف المادة | غير موجود |
| شرح الحل فوراً بعد الإجابة الخاطئة | غير موجود (يظهر فقط في النتائج) |
| تحليل نقاط الضعف حسب المادة | غير موجود |
| دليل الكليات والمتطلبات | غير موجود |

### التغييرات المطلوبة (4 محاور)

---

#### 1. تصنيف الأسئلة حسب المادة (Migration + UI)

**قاعدة البيانات:**
- إضافة عمود `subject` (text) لجدول `questions` مع قيم مثل: `biology`, `chemistry`, `physics`, `math`, `english`, `iq`, `general`
- إضافة عمود `subject` لجدول `exam_attempts.answers` (jsonb) ليحفظ المادة مع كل إجابة

**الملفات المتأثرة:**
- Migration جديد لإضافة العمود
- `AdminContent.tsx` - إضافة حقل اختيار المادة عند إنشاء/تعديل السؤال
- `ExamSimulator.tsx` - عرض اسم المادة فوق كل سؤال أثناء الاختبار

---

#### 2. شرح الحل فوراً أثناء الاختبار

**تعديل `ExamSimulator.tsx`:**
- عند اختيار إجابة خاطئة: إيقاف مؤقت للمؤقت، عرض الإجابة الصحيحة + الشرح لمدة 5 ثوانٍ (أو حتى ينقر "التالي")
- تلوين الخيار الصحيح بالأخضر والخاطئ بالأحمر
- عرض `explanation` في بطاقة أسفل الخيارات
- عند الإجابة الصحيحة: انتقال تلقائي بعد ثانية واحدة مع تأثير بصري إيجابي

---

#### 3. تحليل نقاط الضعف حسب المادة

**تعديل `StudentPerformance.tsx`:**
- استخدام عمود `subject` من الأسئلة لتحليل الأداء حسب المادة
- Radar Chart يعرض النسبة في كل مادة (أحياء، كيمياء، إلخ) بدلاً من الدروس
- بطاقات نصية: "مستواك ممتاز في الكيمياء العضوية (85%) لكنك ضعيف في المتتاليات بالرياضيات (40%)"
- توصيات ذكية: "ننصحك بمراجعة دروس الرياضيات" مع روابط مباشرة

---

#### 4. دليل الكليات والمتطلبات (صفحة جديدة)

**قاعدة البيانات (Migration):**
- إضافة أعمدة لجدول `colleges`:
  - `min_gpa numeric` - الحد الأدنى للمعدل
  - `acceptance_rate numeric` - نسبة القبول
  - `required_documents text[]` - الوثائق المطلوبة
  - `registration_deadline text` - موعد التنسيق
  - `notes text` - ملاحظات إضافية

**صفحة جديدة `CollegeGuide.tsx`:**
- عرض بطاقات الكليات مع فلتر حسب الجامعة
- كل بطاقة تعرض: اسم الكلية، نسبة القبول، الحد الأدنى، الوثائق، مواعيد التنسيق
- بحث نصي سريع
- إضافة Route جديد `/college-guide`
- رابط في Dashboard والقائمة الجانبية

**تحديث `AdminColleges.tsx`:**
- إضافة حقول التعديل للبيانات الجديدة (المعدل، النسبة، الوثائق، المواعيد)

---
