
# نظام Freemium وطرق الدفع الإلكترونية

## 1. توسيع أنواع طرق الدفع
- تعديل دالة التحقق `validate_payment_method_type` لتقبل نوع `ewallet` بجانب `bank` و `exchange`
- هذا يتيح للأدمن إضافة أي محفظة إلكترونية يمنية (جوالي، فلوسك، كاش، إلخ) من لوحة التحكم

## 2. نظام Freemium للدروس
- تعديل سياسة RLS للدروس: السماح لأي مستخدم مسجل برؤية **ملخصات الدروس** (عنوان + ملخص فقط)
- المحتوى الكامل والأسئلة تبقى محصورة بالمشتركين
- إنشاء سياسة جديدة تفصل بين البيانات الأساسية والمحتوى الكامل

## 3. اختبار تجريبي مجاني
- إضافة عمود `is_free_trial` إلى جدول `lessons` لتحديد الدروس المجانية
- السماح بدخول اختبار واحد مجاناً (أول درس مجاني لكل تخصص)

## 4. تعديل واجهة الدروس
- عرض الملخصات مع أيقونة 🔒 للمحتوى المقفل
- زر "اشترك الآن" للوصول للمحتوى الكامل
- تعديل صفحة الدفع لعرض خيار المحافظ الإلكترونية
